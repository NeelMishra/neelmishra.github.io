    Linear Regression: Foundations

    July 2026 · 18 min read

    Linear regression predicts a numeric target by adding weighted feature values. It is simple enough to solve exactly, but deep enough to introduce the core language of supervised learning: model, loss, residual, optimization, assumptions, regularization, and diagnostics. It is also the baseline that more complex models must beat.

    The Model

    For one feature, the model is a line:

    $$\hat y = b + wx.$$

    For many features, it is a weighted sum:

    $$\hat y = b + w_1x_1 + w_2x_2 + \cdots + w_dx_d.$$

    The word "linear" refers to linearity in the parameters, not necessarily to raw features. Polynomial regression is still linear regression if the features are $x$, $x^2$, and $x^3$ and the model is linear in their weights.

    Residuals and Least Squares

    The residual for row $i$ is $e_i = y_i-\hat y_i$. Ordinary least squares chooses parameters that minimize the sum of squared residuals:

    $$\min_{b,w}\sum_i (y_i-b-w^\top x_i)^2.$$

    Squaring makes large errors expensive, gives a smooth objective, and leads to a closed-form solution when the design matrix has enough independent information.

    Why Linear Regression Still Matters

    • Interpretability: each coefficient is a marginal effect under the fitted linear model.
    • Optimization: ordinary least squares is convex, so local minima are not a problem.
    • Diagnostics: residual plots expose nonlinearity, outliers, heteroscedasticity, and leakage.
    • Baseline value: if a complex model only barely beats linear regression, the extra complexity may not be worth it.
