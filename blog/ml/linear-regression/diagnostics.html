    Linear Regression Assumptions and Diagnostics

    July 2026 · 15 min read

    Linear regression is useful because its failure modes are visible. Residuals should look like unstructured noise around zero. When patterns remain, the model is telling that the linear story is incomplete.

    SymptomLikely issuePossible repair
    Curved residual patternNonlinear relationshipAdd transformations, polynomial terms, splines, or use a nonlinear model.
    Residual spread grows with fitted valueHeteroscedasticityTransform target, use weighted least squares, or model variance separately.
    One row dominates the fitOutlier or high leverageAudit the row, use robust regression, or cap invalid values.
    Coefficients swing wildlyMulticollinearityUse Ridge, remove redundant features, or interpret grouped effects.

    Validation

    Good diagnostics are not only in-sample. A model with beautiful training residuals can still fail on test data if features leak future information or preprocessing was fit on the full dataset. Split first, fit preprocessing on train only, and evaluate on held-out data.

    The residual plot is the first debugging tool. It often shows whether to add features, transform variables, regularize, or leave linear regression entirely.
