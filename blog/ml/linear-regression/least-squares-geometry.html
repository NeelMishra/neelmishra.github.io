    Least Squares Geometry

    July 2026 · 17 min read

    In matrix form, linear regression writes predictions as $\hat y = Xw$. The columns of $X$ span all prediction vectors the model can produce. Least squares finds the point in that column space closest to the target vector $y$.

    Projection View

    The fitted vector $\hat y$ is the orthogonal projection of $y$ onto the column space of $X$. The residual vector $r=y-\hat y$ is perpendicular to every column of $X$:

    $$X^\top(y-Xw)=0.$$

    Rearranging gives the normal equations:

    $$X^\top Xw = X^\top y.$$

    ▶ Projection onto the Model Space

    The target vector is projected onto the model's column space. The residual is orthogonal to the fitted vector.

    Rank and Multicollinearity

    If the columns of $X$ are linearly independent, $X^\top X$ is invertible and the normal-equation solution is unique. If columns are redundant, many coefficient vectors can produce the same fitted values. This is multicollinearity: predictions may be stable, but individual coefficients become unstable and hard to interpret.

    Numerical Stability

    Solving normal equations directly squares the condition number of the design matrix. QR decomposition or SVD is usually more stable. In from-scratch educational code, normal equations are useful because they show the algebra clearly, but production numerical code should prefer stable factorizations.

    Core geometry. Least squares is not magic curve fitting. It is nearest-point projection in a vector space of possible predictions.
