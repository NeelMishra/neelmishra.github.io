    PCR and PLS

    July 2026 · 15 min read

    When many predictors move together, their separate coefficients can become unstable. PCR and PLS replace the original columns with a smaller set of components.

    A component is a weighted sum of original predictors. The weights are called loadings, and each row's value on that component is its score. The methods differ in how they choose those weighted sums:

    • Principal component regression (PCR) chooses directions with large predictor variation and looks at the response only afterward.
    • Partial least squares (PLS) uses the response while choosing directions.

    Both methods can help when there are many correlated predictors and a lower-dimensional signal is plausible. Neither automatically selects a small set of original features, so coefficient interpretation can remain difficult.

    Why the Largest Predictor Direction Can Miss the Response

    The centered rows are $(-3,-1),(-1,1),(1,1),(3,-1)$ with $y=x_2$. The states derive predictor sums of squares, PCR projections, and the response-aware PLS direction.

    Principal Component Regression

    After centering the predictors, write the singular value decomposition as

    $$X=UDV^\top.$$
    • The columns of $V$ are predictor directions, or loading vectors.
    • The singular values in $D$ measure how much predictor variation lies in each direction.
    • The score matrix is $XV=UD$.

    PCR keeps the first $k$ directions with the largest singular values:

    $$Z_k=XV_k=U_kD_k.$$

    It regresses $y$ on $Z_k$ to obtain score coefficients $\hat\gamma$, then maps them back to original coordinates as $\hat\beta=V_k\hat\gamma$. Discarding small-singular-value directions removes unstable coefficient directions.

    Predictor variation is not response relevance. PCR never uses $y$ while building components. A low-variance predictor direction can be the direction most strongly related to the response.

    Partial Least Squares

    PLS chooses a loading direction whose score has high covariance with $y$. After one score is built, deflation removes the predictor variation explained by that score, and the next component is found from what remains.

    Because the response guides the directions, PLS can find a predictive low-variance direction before PCR. It can also overfit more easily because every component has seen $y$. PLS algorithms differ in normalization and deflation details, so the fitted variant must be stated.

    Standardization and Component Tuning

    Component methods are sensitive to units. A predictor measured in dollars can have much larger variance than one measured in years and dominate the directions for that reason alone. Centering and scaling usually put columns on comparable units, though domain meaning can justify a different choice.

    The complete pipeline is fitted inside each training fold:

    1. Missing-value rules, centers, and scales are learned from the fold's training rows.
    2. PCR or PLS loadings are learned from those same rows.
    3. The number of retained components is selected by validation performance.

    Building components on the full dataset leaks held-out predictor structure, even for response-blind PCR. PLS can leak both predictor and response information.

    Few components create bias by deleting directions. More components recover signal but can restore unstable low-variance directions and noise. Cross-validation measures this tradeoff for the actual prediction target.

    Interview distinction. PCR is unsupervised dimension reduction followed by regression. PLS is supervised dimension reduction because $y$ influences the score directions.

    Two-Direction Counterexample

    The centered rows and response are

    $$X=\{(-3,-1),(-1,1),(1,1),(3,-1)\},\qquad y=x_2=(-1,1,1,-1).$$

    The two columns are orthogonal because $x_1^\top x_2=0$. Their sums of squares are

    $$x_1^\top x_1=20,\qquad x_2^\top x_2=4.$$

    One-component PCR selects $x_1$ because it contains $20/(20+4)=83.33\%$ of predictor variation. But

    $$x_1^\top y=0,$$

    so that component has no predictive signal. PLS instead sees

    $$X^\top y=(0,4)^\top,$$

    selects the $x_2$ direction, and predicts $y$ exactly in this noiseless example. Most predictor variation was not the useful variation.

    Comparison with Ridge and Lasso

    MethodHow it controls instabilityMain interpretation
    PCRSets every discarded singular direction exactly to zeroHard truncation in component space
    RidgeMultiplies each OLS direction by $d_j^2/(d_j^2+\lambda)$Continuous shrinkage in every singular direction
    LassoCan set original-feature coefficients to zeroSparse original-feature model, with unstable selection among correlated features

    In direct SVD form, ridge uses $d_j/(d_j^2+\lambda)$ on the response projection $u_j^\top y$. Relative to the OLS coefficient in that direction, the shrinkage factor is $d_j^2/(d_j^2+\lambda)$.

    Diagnostics and Edge Cases

    • Component count. Held-out prediction selects $k$; cumulative predictor variance alone does not.
    • Loading sign. Multiplying a loading and its score by $-1$ represents the same component.
    • Repeated singular values. Individual directions can rotate within a tied subspace, although the subspace remains defined.
    • Missing values. Imputation is learned separately inside each training fold.
    • PLS flexibility. Too many response-aware components can fit noise.

    SVD and conditioning details appear in reliable least-squares solvers. Continuous spectral shrinkage appears in regularization.

    Interview Quick Checks

    PromptComplete answer
    PCR versus PCA?PCA produces predictor scores. PCR adds a regression of the response on a selected subset of those scores.
    PCR versus PLS?PCR ranks directions by predictor variance, while PLS constructs directions using association with the response.
    Why can PCR fail?A predictive direction can have low predictor variance and therefore appear late in the principal-component ordering.
    Where does leakage occur?Scaling, component construction, and component-count selection all leak information when fitted outside each training fold.
