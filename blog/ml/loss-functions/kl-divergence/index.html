    KL Divergence Foundations

    July 2026 · 18 min read

    Kullback-Leibler divergence measures how much a model distribution $Q$ wastes, on average, when the data really follows $P$. It is also called relative entropy:

    $$D_{KL}(P\|Q)=\sum_i p_i\log_2\frac{p_i}{q_i}.$$

    It is zero only when the two distributions match on the support of $P$. It is not symmetric, so the order matters.

    Excess Code Length

    Entropy is the best possible average code length for data from $P$. Cross-entropy is the average code length paid when the code is designed for $Q$. The difference is KL divergence:

    $$H(P,Q)-H(P)=D_{KL}(P\|Q).$$

    ▶ KL as Wasted Bits

    The true distribution P stays fixed. Step through model distributions Q. The wasted bits shrink as Q approaches P.

    Direction Matters

    $D_{KL}(P\|Q)$ punishes $Q$ harshly when it assigns tiny probability to events that actually occur under $P$. Reverse KL, $D_{KL}(Q\|P)$, has a different behavior and often prefers concentrating on one high-probability mode. The next post focuses on that directional difference.

    KL is not a distance metric. It is nonnegative, but it is not symmetric and does not satisfy the triangle inequality.
