    Jensen-Shannon Divergence Foundations

    July 2026 · 16 min read

    Jensen-Shannon divergence turns KL into a symmetric, smoothed comparison. Given two distributions $P$ and $Q$, form the midpoint mixture $M=\tfrac12(P+Q)$, then compute

    $$JS(P,Q)=\tfrac12D_{KL}(P\|M)+\tfrac12D_{KL}(Q\|M).$$

    Both distributions are compared to a shared mixture, so the result is symmetric and bounded. With log base 2, JS lies between $0$ and $1$ bit.

    ▶ JS Compares Both Distributions to the Mixture

    The midpoint distribution M smooths the comparison. Each side pays KL to the midpoint, then the two costs are averaged.

    Why JS Is Useful

    KL can become infinite when the model assigns zero probability where the data has mass. JS avoids this by comparing both distributions to the mixture. It is also symmetric, so it is easier to use as a descriptive distribution difference.

    JS is still not the whole story. The square root of JS is a metric, but JS itself can saturate when distributions have little overlap. That saturation is one reason Wasserstein distance became popular in generative modeling.
