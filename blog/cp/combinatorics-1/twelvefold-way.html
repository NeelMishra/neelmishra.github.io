    The Twelvefold Way: A Systematic Framework

    September 2026 · 22 min read

    What Is the Twelvefold Way

    The Twelvefold Way is a classification scheme introduced by Gian-Carlo Rota that organizes virtually every basic counting problem into a 4 × 3 grid. It asks a single question in two dimensions:

    • Rows — what restriction do we place on the function f : N → X?
      • Unrestricted (any function)
      • Injective (at most one ball per box)
      • Surjective (every box gets at least one ball)
    • Columns — which objects are distinguishable?
      • Both N (balls) and X (boxes) are labeled
      • Balls unlabeled, boxes labeled
      • Balls labeled, boxes unlabeled
      • Both unlabeled

    This gives 3 × 4 = 12 cases — the "twelvefold way." Each cell maps to a well-known formula or sequence in combinatorics.

    Why memorize this? In competitive programming, recognizing which of the 12 cases a problem falls into immediately tells you the formula. No more re-deriving from scratch under time pressure.

    The 12 Cases — Summary Table

    Let |N| = n (number of balls) and |X| = x (number of boxes). The following table summarizes all 12 cases.

    f restriction N labeled, X labeled N unlabeled, X labeled N labeled, X unlabeled N unlabeled, X unlabeled
    Any f xn C(n+x−1, n)
    (stars & bars)    		 Σ S(n,k) for k=1..x
    (sum of Stirling 2nd)    		 Σ p(n,k) for k=1..x
    (partitions of n into ≤ x parts)
    Injective x! / (x−n)!
    = P(x,n)    		 C(x, n) [n ≤ x] · 1 [n ≤ x] · 1
    Surjective x! · S(n, x)
    (surjections via PIE)    		 C(n−1, x−1)
    (compositions)    		 S(n, x)
    (Stirling 2nd kind)    		 p(n, x)
    (partitions into exactly x parts)

    Here S(n,k) denotes Stirling numbers of the second kind, p(n,k) denotes the number of partitions of n into exactly k parts, and C(a,b) is the binomial coefficient.

    Detailed Analysis of Each Case

    Case 1: Any f, both labeled → xn

    Each of the n labeled balls independently chooses one of x labeled boxes. Total: x choices for each of n balls = xn.

    Example: How many strings of length n over an alphabet of size x? Answer: xn.

    Case 2: Injective f, both labeled → P(x, n)

    Each ball goes to a distinct box. The first ball has x choices, the second x−1, etc. Total: x · (x−1) · … · (x−n+1) = x! / (x−n)!. This requires n ≤ x.

    Example: Seating n people in x chairs (n ≤ x), each in a distinct chair.

    Case 3: Surjective f, both labeled → x! · S(n, x)

    Every box must receive at least one ball. This counts surjections from N to X. By inclusion-exclusion:

    surj(n, x) = Σj=0x (−1)j C(x, j) (x − j)n

    This equals x! · S(n, x) where S(n, x) is the Stirling number of the second kind.

    Case 4: Any f, balls unlabeled, boxes labeled → C(n+x−1, n)

    We only care how many balls each box gets. This is the classic stars and bars problem: distribute n identical items into x distinct bins. Answer: C(n + x − 1, n).

    Case 5: Injective f, balls unlabeled, boxes labeled → C(x, n)

    At most one (identical) ball per box. We simply choose which n of the x boxes are occupied: C(x, n).

    Case 6: Surjective f, balls unlabeled, boxes labeled → C(n−1, x−1)

    Each box must get at least one ball. Place one ball in each box first (costing x balls), then distribute the remaining n − x identical balls into x boxes: C((n−x) + x − 1, n−x) = C(n − 1, x − 1). This is also the compositions of n into exactly x positive parts.

    Case 7: Any f, balls labeled, boxes unlabeled → Σ S(n,k), k=1..x

    Boxes are indistinguishable, so we partition the set of n labeled balls into at most x non-empty subsets. The count is Σk=1min(n,x) S(n, k), sometimes written Bx(n) (a truncated Bell number).

    Case 8: Injective f, balls labeled, boxes unlabeled → [n ≤ x]

    Each box gets at most one ball, and boxes are indistinguishable. If n ≤ x, there is exactly one way (every ball is alone in its box, and the boxes have no labels to permute). If n > x, impossible.

    Case 9: Surjective f, balls labeled, boxes unlabeled → S(n, x)

    Partition n labeled balls into exactly x non-empty indistinguishable groups. This is the definition of the Stirling number of the second kind S(n, x).

    Case 10: Any f, both unlabeled → partitions of n into ≤ x parts

    Distribute n identical balls into x identical boxes (some may be empty). This counts integer partitions of n with at most x parts.

    Case 11: Injective f, both unlabeled → [n ≤ x]

    Identical balls into identical boxes, at most one per box. Exactly one way if n ≤ x; impossible otherwise.

    Case 12: Surjective f, both unlabeled → p(n, x)

    Distribute n identical balls into x identical boxes, each non-empty. This counts integer partitions of n into exactly x parts, denoted p(n, x).

    Memory aid: "Labeled" means order/identity matters; "unlabeled" means only quantity matters. For balls, labeled = distinguishable items, unlabeled = identical items. For boxes, labeled = distinct containers, unlabeled = interchangeable containers.

    Connection to Known Sequences

    Each cell of the Twelvefold Way connects to a well-studied combinatorial sequence:

    • xn — words / functions / sequences
    • P(x, n) = x!/(x−n)! — permutations / arrangements
    • C(x, n) — binomial coefficients / combinations
    • C(n+x−1, n) — multiset coefficients / stars and bars
    • C(n−1, x−1) — compositions
    • S(n, k) — Stirling numbers of the second kind / set partitions
    • B(n) — Bell numbers (sum of all S(n, k))
    • p(n) — integer partitions
    • p(n, k) — restricted partitions
    • Surjections — counted via inclusion-exclusion

    Mastering the table means you instantly recognize which formula to apply when you read a problem statement.

    When to Use Which Formula

    Here is a decision flowchart for competitive programming:

    1. Identify the "balls" and "boxes" — what is being distributed? Into what?
    2. Are the balls labeled or unlabeled? — Are the objects distinguishable?
    3. Are the boxes labeled or unlabeled? — Are the containers distinguishable?
    4. What restriction on f?
      • No restriction → "any f" row
      • "At most one item per container" → injective row
      • "Every container must be non-empty" → surjective row
    5. Look up the cell in the table above.
    Common mistake: Confusing "labeled boxes" with "unlabeled boxes." If the problem says "distribute into groups" without naming or ordering the groups, boxes are unlabeled. If it says "distribute into bins 1, 2, …, x" or the boxes are people/slots, they are labeled.

    C++ Utilities for Each Case

    #include <bits/stdc++.h>
using namespace std;

const int MOD = 1e9 + 7;
const int MAXN = 2000005;

long long fac[MAXN], ifac[MAXN];

long long power(long long a, long long b, long long mod = MOD) {
    long long r = 1; a %= mod;
    while (b > 0) { if (b & 1) r = r * a % mod; a = a * a % mod; b >>= 1; }
    return r;
}

void precompute() {
    fac[0] = 1;
    for (int i = 1; i < MAXN; i++) fac[i] = fac[i - 1] * i % MOD;
    ifac[MAXN - 1] = power(fac[MAXN - 1], MOD - 2);
    for (int i = MAXN - 2; i >= 0; i--) ifac[i] = ifac[i + 1] * (i + 1) % MOD;
}

long long C(int n, int r) {
    if (r < 0 || r > n) return 0;
    return fac[n] % MOD * ifac[r] % MOD * ifac[n - r] % MOD;
}

// Case 1: Any f, both labeled: x^n
long long case1(int n, int x) { return power(x, n); }

// Case 2: Injective, both labeled: P(x, n)
long long case2(int n, int x) {
    if (n > x) return 0;
    return fac[x] % MOD * ifac[x - n] % MOD;
}

// Case 3: Surjective, both labeled: x! * S(n, x) via PIE
long long surjections(int n, int x) {
    long long ans = 0;
    for (int j = 0; j <= x; j++) {
        long long term = C(x, j) * power(x - j, n) % MOD;
        if (j & 1) ans = (ans - term + MOD) % MOD;
        else        ans = (ans + term) % MOD;
    }
    return ans;
}

// Case 4: Any f, balls unlabeled, boxes labeled: C(n+x-1, n)
long long case4(int n, int x) { return C(n + x - 1, n); }

// Case 5: Injective, balls unlabeled, boxes labeled: C(x, n)
long long case5(int n, int x) { return C(x, n); }

// Case 6: Surjective, balls unlabeled, boxes labeled: C(n-1, x-1)
long long case6(int n, int x) {
    if (n < x) return 0;
    return C(n - 1, x - 1);
}

// Case 9: Surjective, balls labeled, boxes unlabeled: S(n, x)
long long stirling2(int n, int x) {
    // S(n,x) = surjections(n,x) / x!
    return surjections(n, x) % MOD * ifac[x] % MOD;
}

// Case 7: Any f, balls labeled, boxes unlabeled: sum S(n,k) for k=1..x
long long case7(int n, int x) {
    long long ans = 0;
    for (int k = 1; k <= min(n, x); k++)
        ans = (ans + stirling2(n, k)) % MOD;
    return ans;
}
    Note on partition counting (Cases 10, 12): Integer partitions do not have a simple closed-form and are typically computed via DP. A standard O(n · x) DP fills a table where dp[i][j] = number of partitions of i into at most j parts.

    Partition DP (Cases 10 & 12)

    // dp[i][j] = number of partitions of i into at most j parts
// For "exactly j parts," use dp[i][j] - dp[i][j-1]
vector<vector<long long>> partition_table(int n, int x) {
    vector<vector<long long>> dp(n + 1, vector<long long>(x + 1, 0));
    for (int j = 0; j <= x; j++) dp[0][j] = 1; // empty partition
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= x; j++) {
            dp[i][j] = dp[i][j - 1]; // don't use part of size j
            if (i >= j) dp[i][j] = (dp[i][j] + dp[i - j][j]) % MOD;
        }
    return dp;
}

// Case 10: Any f, both unlabeled: partitions of n into <= x parts
long long case10(int n, int x) {
    auto dp = partition_table(n, x);
    return dp[n][x];
}

// Case 12: Surjective, both unlabeled: partitions of n into exactly x parts
long long case12(int n, int x) {
    if (n < x) return 0;
    auto dp = partition_table(n, x);
    return (dp[n][x] - dp[n][x - 1] + MOD) % MOD;
}

    Worked Examples

    Example 1: Distributing Prizes

    10 distinct prizes are given to 4 distinct people. Each person must get at least one prize. How many ways?

    Balls = prizes (labeled), boxes = people (labeled), surjective. This is Case 3: surjections(10, 4) = 4! · S(10, 4).

    // Answer: surjections(10, 4)
// = C(4,0)*4^10 - C(4,1)*3^10 + C(4,2)*2^10 - C(4,3)*1^10
// = 1048576 - 236196 + 6144 - 4 = 818520

    Example 2: Distributing Identical Coins

    Place 7 identical coins into 3 labeled jars with no restrictions.

    Balls unlabeled, boxes labeled, any f. Case 4: C(7 + 3 − 1, 7) = C(9, 7) = 36.

    Example 3: Partitioning a Set

    How many ways to partition {1,2,3,4,5} into exactly 3 non-empty groups?

    Balls labeled, boxes unlabeled, surjective. Case 9: S(5, 3) = 25.

    Example 4: Integer Partitions

    How many ways to write 10 as a sum of exactly 3 positive integers (order doesn't matter)?

    Balls unlabeled, boxes unlabeled, surjective. Case 12: p(10, 3) = 8.

    The partitions: 1+1+8, 1+2+7, 1+3+6, 1+4+5, 2+2+6, 2+3+5, 2+4+4, 3+3+4.

    Practice Problems

    Test your mastery of the Twelvefold Way:

    1. Balls in Boxes I (AtCoder ABC 132E) — Direct application of stars and bars (Case 4 or 6).
    2. Distributing Distinct Items (CF 1342E) — Surjections with labeled boxes. Apply inclusion-exclusion.
    3. Set Partition Counting — Compute S(n, k) mod p for n, k up to 105. Use the PIE formula with precomputed factorials.
    4. Integer Partition Counting (Project Euler 76) — How many ways to write 100 as a sum of at least two positive integers? Use the partition DP.
    5. Necklace Distribution — Give n identical beads to k identical children (each gets at least one). Recognize this as Case 12 and compute p(n, k).
    6. Surjection Counting (CF 886E) — Count surjections modulo a prime. PIE + modular arithmetic.
    7. Mixed Problem — A problem with multiple phases; each phase corresponds to a different cell of the Twelvefold Way. Combine the counts multiplicatively.
    Study strategy: For each of the 12 cells, find and solve at least one problem. This ensures you can recognize each case and apply the correct formula under contest conditions.
