    Trie Operations

    March 2026 · 22 min read

    We know the trie structure. Now let's implement every operation, see it animated, and understand the time complexity of each.

    Insert

    Walk the string character by character. At each step, if the child for that character doesn't exist, create it. After processing the last character, mark the node as end-of-word.

    void insert(TrieNode* root, const string& word) {
    TrieNode* cur = root;
    for (char c : word) {
        int idx = c - 'a';
        if (!cur->children[idx])
            cur->children[idx] = new TrieNode();
        cur = cur->children[idx];
    }
    cur->isEnd = true;
}

    Time: O(m) where m is the word length. We touch exactly m nodes.

    Search (Exact Match)

    Walk the string character by character. If at any point the child is null, the word doesn't exist. If we reach the end, check the isEnd flag.

    bool search(TrieNode* root, const string& word) {
    TrieNode* cur = root;
    for (char c : word) {
        int idx = c - 'a';
        if (!cur->children[idx]) return false;  // path doesn't exist
        cur = cur->children[idx];
    }
    return cur->isEnd;  // reached end, but is it a complete word?
}

    Important: reaching the last character node is not enough. If isEnd is false, the string is only a prefix, not a stored word. Example: if only "apple" is stored, searching for "app" reaches the second 'p' node but isEnd is false.

    Prefix Search & Autocomplete

    This is the trie's superpower. To find all words starting with a prefix:

    1. Walk the prefix to reach its node (same as search, but don't check isEnd).
    2. From that node, do a DFS collecting all end-of-word nodes.
    vector<string> autocomplete(TrieNode* root, const string& prefix) {
    TrieNode* cur = root;
    for (char c : prefix) {
        if (!cur->children[c - 'a']) return {};  // prefix doesn't exist
        cur = cur->children[c - 'a'];
    }
    vector<string> results;
    dfs(cur, prefix, results);
    return results;
}

void dfs(TrieNode* node, string& path, vector<string>& results) {
    if (node->isEnd) results.push_back(path);
    for (int i = 0; i < 26; i++) {
        if (node->children[i]) {
            path.push_back('a' + i);
            dfs(node->children[i], path, results);
            path.pop_back();
        }
    }
}

    Time: O(p + k · L) where p is prefix length, k is number of matching words, L is average word length. The DFS touches only the subtree below the prefix node.

    Starts With (Prefix Exists)

    A simpler variant: just check if any word starts with the given prefix. Same traversal as search, but return true if we reach the end of the prefix (regardless of isEnd).

    bool startsWith(TrieNode* root, const string& prefix) {
    TrieNode* cur = root;
    for (char c : prefix) {
        if (!cur->children[c - 'a']) return false;
        cur = cur->children[c - 'a'];
    }
    return true;  // prefix exists (may or may not be a complete word)
}

    Delete

    Deletion is the trickiest operation. We want to remove a word without accidentally removing prefixes of other words.

    bool remove(TrieNode* node, const string& word, int depth) {
    if (depth == word.size()) {
        if (!node->isEnd) return false;  // word not found
        node->isEnd = false;
        return isEmpty(node);  // true if node has no children (can be freed)
    }
    int idx = word[depth] - 'a';
    if (!node->children[idx]) return false;  // word not found

    bool shouldDelete = remove(node->children[idx], word, depth + 1);

    if (shouldDelete) {
        delete node->children[idx];
        node->children[idx] = nullptr;
        return !node->isEnd && isEmpty(node);  // cascade if node is now useless
    }
    return false;
}

    The key logic: after unmarking isEnd, check if the node has any children. If not, it's a dead-end node that should be freed. This cleanup cascades up the recursion, deleting any ancestor nodes that are also now childless and not end-of-word.

    Count Words With Prefix

    A common optimization: store a count at each node tracking how many words pass through it. This lets you answer "how many words start with X?" in O(m) without DFS.

    struct TrieNode {
    TrieNode* children[26];
    int wordCount;   // words that end here
    int prefixCount; // words that pass through here
};

void insert(TrieNode* root, const string& word) {
    TrieNode* cur = root;
    for (char c : word) {
        int idx = c - 'a';
        if (!cur->children[idx])
            cur->children[idx] = new TrieNode();
        cur = cur->children[idx];
        cur->prefixCount++;
    }
    cur->wordCount++;
}

    Complexity Summary

    OperationTimeSpace per op
    InsertO(m)O(m) new nodes worst case
    SearchO(m)O(1)
    startsWithO(p)O(1)
    AutocompleteO(p + k·L)O(k·L) for results
    DeleteO(m)O(1)
    Count prefixO(p)O(1)

    m = word length, p = prefix length, k = number of matching words, L = average word length
