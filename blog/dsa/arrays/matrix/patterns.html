    Matrix Patterns

    March 2026 · 22 min read

    This post covers four classic matrix problems that appear frequently in interviews. Each one has a clean pattern that, once understood, makes the solution straightforward.

    Rotate Matrix 90 Degrees Clockwise

    Given an n x n matrix, rotate it 90 degrees clockwise in place. This is LeetCode 48.

    The trick is to break it into two steps:

    1. Transpose the matrix (swap mat[i][j] with mat[j][i]).
    2. Reverse each row (mirror horizontally).

    Why does this work? A 90-degree clockwise rotation maps element (i, j) to position (j, n-1-i). Transposing maps (i, j) to (j, i). Reversing each row then maps (j, i) to (j, n-1-i). Combining these two steps gives exactly the rotation mapping.

    Pattern: Rotate clockwise = Transpose + Reverse each row. Rotate counter-clockwise = Transpose + Reverse each column (or Reverse each row + Transpose).

    Step through to see the rotation of a 4x4 matrix. Colored pairs show elements swapping positions.

    Original Matrix
    Mapping
    Press Step to see each phase. 
    // Rotate matrix 90 degrees clockwise (in place)
void rotate(vector<vector<int>>& mat) {
    int n = mat.size();

    // Step 1: Transpose
    for (int i = 0; i < n; i++)
        for (int j = i + 1; j < n; j++)
            swap(mat[i][j], mat[j][i]);

    // Step 2: Reverse each row
    for (int i = 0; i < n; i++)
        reverse(mat[i].begin(), mat[i].end());
}
// Time: O(n^2), Space: O(1)

    Other Rotation Directions

    Counter-clockwise 90: Transpose, then reverse each column. Or equivalently: reverse each row, then transpose.

    180 degrees: Reverse each row, then reverse each column. Or rotate 90 twice.

    Search in a Row-Column Sorted Matrix

    Given a matrix where each row is sorted left to right and each column is sorted top to bottom, find whether a target value exists. This is LeetCode 240.

    The key insight is the staircase search. Start at the top-right corner. From there:

    • If the current value equals the target, return true.
    • If the current value is greater than the target, move left (eliminate the current column since everything below is even larger).
    • If the current value is less than the target, move down (eliminate the current row since everything to the left is even smaller).

    Each step eliminates either a full row or a full column. Since the matrix has m rows and n columns, the search takes at most m + n steps.

    Searching for value 14 in a 4x5 sorted matrix. Green = current cell, orange = eliminated area.

    Press Step to begin. Target = 14. 
    // Staircase search in row-column sorted matrix
bool searchMatrix(vector<vector<int>>& mat, int target) {
    if (mat.empty()) return false;
    int m = mat.size(), n = mat[0].size();
    int r = 0, c = n - 1;  // Start at top-right

    while (r < m && c >= 0) {
        if (mat[r][c] == target) return true;
        else if (mat[r][c] > target) c--;  // Eliminate column
        else r++;                           // Eliminate row
    }
    return false;
}
// Time: O(m + n), Space: O(1)

    Starting Corner Choice

    Top-right and bottom-left both work because they sit at a "decision point": one direction has larger values, the other has smaller. Top-left and bottom-right do not work because both directions go the same way (both increasing or both decreasing).

    Set Matrix Zeroes

    Given an m x n matrix, if an element is 0, set its entire row and entire column to 0. Do it in place. This is LeetCode 73.

    The naive approach uses O(m + n) space to record which rows and columns contain zeroes. The optimal approach uses the first row and first column as markers, achieving O(1) extra space.

    Algorithm

    1. Check if the first row and first column themselves contain any zeroes (store this in two boolean flags).
    2. Scan the rest of the matrix. For each zero at (i, j), mark mat[i][0] = 0 and mat[0][j] = 0.
    3. Use the markers in the first row and column to zero out the corresponding rows and columns (process rows 1 to m-1 and columns 1 to n-1).
    4. Finally, if the first row flag was set, zero out the entire first row. Same for first column.

    Walk through the zero-marking process on a 4x4 matrix. Red = zero, green marker = flagged row/column header.

    Press Step to begin. 
    // Set Matrix Zeroes - O(1) space
void setZeroes(vector<vector<int>>& mat) {
    int m = mat.size(), n = mat[0].size();
    bool firstRowZero = false, firstColZero = false;

    // Check if first row has zero
    for (int j = 0; j < n; j++)
        if (mat[0][j] == 0) firstRowZero = true;

    // Check if first column has zero
    for (int i = 0; i < m; i++)
        if (mat[i][0] == 0) firstColZero = true;

    // Mark zeroes in first row/column
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            if (mat[i][j] == 0) {
                mat[i][0] = 0;
                mat[0][j] = 0;
            }

    // Zero out cells based on markers
    for (int i = 1; i < m; i++)
        for (int j = 1; j < n; j++)
            if (mat[i][0] == 0 || mat[0][j] == 0)
                mat[i][j] = 0;

    // Handle first row
    if (firstRowZero)
        for (int j = 0; j < n; j++) mat[0][j] = 0;

    // Handle first column
    if (firstColZero)
        for (int i = 0; i < m; i++) mat[i][0] = 0;
}
// Time: O(m * n), Space: O(1)

    Order Matters

    You must process the first row and first column last. If you zero them out early, you lose the marker information needed to decide which inner rows and columns to clear.

    Matrix Multiplication Basics

    Multiplying two matrices A (size m x k) and B (size k x n) produces a result C (size m x n). Each element C[i][j] is the dot product of row i of A and column j of B:

    C[i][j] = A[i][0]*B[0][j] + A[i][1]*B[1][j] + ... + A[i][k-1]*B[k-1][j]

    The inner dimension must match: A has k columns, B has k rows. The result takes the outer dimensions: m rows from A, n columns from B.

    Multiplying a 2x3 matrix by a 3x2 matrix. Watch which row and column combine to produce each output cell.

    Press Step to begin. 
    // Matrix multiplication: A (m x k) * B (k x n) = C (m x n)
vector<vector<int>> multiply(vector<vector<int>>& A,
                              vector<vector<int>>& B) {
    int m = A.size(), k = A[0].size(), n = B[0].size();
    vector<vector<int>> C(m, vector<int>(n, 0));

    for (int i = 0; i < m; i++)
        for (int j = 0; j < n; j++)
            for (int p = 0; p < k; p++)
                C[i][j] += A[i][p] * B[p][j];

    return C;
}
// Time: O(m * k * n), Space: O(m * n)

    Loop Order for Cache Performance

    The loop order i, p, j (instead of i, j, p) is more cache-friendly. It accesses B row-by-row instead of jumping across columns. For large matrices, this can be 2 to 10 times faster due to fewer cache misses.

    // Cache-friendly loop order
for (int i = 0; i < m; i++)
    for (int p = 0; p < k; p++)
        for (int j = 0; j < n; j++)
            C[i][j] += A[i][p] * B[p][j];

    Sparse Matrix Multiplication

    When matrices contain many zeroes, skip zero entries to avoid wasted multiplications:

    // Sparse multiplication: skip zero elements in A
for (int i = 0; i < m; i++)
    for (int p = 0; p < k; p++) {
        if (A[i][p] == 0) continue;  // Skip zero
        for (int j = 0; j < n; j++)
            C[i][j] += A[i][p] * B[p][j];
    }

    Pattern Summary

    ProblemKey IdeaTimeSpace
    Rotate 90 CWTranspose + reverse rowsO(n^2)O(1)

    Set matrix zeroesUse first row/col as markersO(m * n)O(1)

    Summary

    • Rotating a matrix 90 degrees clockwise decomposes into transpose plus row reversal. No extra space needed.
    • Staircase search eliminates one row or column per step, achieving O(m + n) time in a row-column sorted matrix.
    • Set matrix zeroes optimally by using the first row and column as in-place markers, processing them last.
    • Matrix multiplication runs in O(m * k * n). Reorder loops to i, p, j for better cache performance.
    • In interviews, always clarify whether the matrix is square or rectangular, and whether in-place modification is required.
