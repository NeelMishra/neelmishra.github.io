    Linear Regression from Scratch in C++

    July 2026 · Series overview

    This series implements the Linear Regression deep dive in plain C++, using only the standard library. It starts with ordinary least squares, then moves to iterative optimization, regularized variants, and robust losses.

    Implementation Roadmap

    Ordinary Least Squares

    Build the design matrix, solve normal equations with Gaussian elimination, and report RMSE.

    Gradient Descent

    Train the same linear model with batch and stochastic updates.

    Regularized Regression

    Implement Ridge in closed form and Lasso with coordinate descent.

    Robust and Quantile Regression

    Swap squared error for Huber and pinball losses.
