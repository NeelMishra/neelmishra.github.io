    Gradient Descent Linear Regression in C++

    July 2026 · 15 min read

    Gradient descent avoids forming and solving normal equations. It is useful for large data, online updates, and losses that do not have a closed form.

    Vector fit_gd(const Matrix& X, const Vector& y, int epochs, double lr) {
    int n = (int)X.size();
    int d = (int)X[0].size() + 1;
    Vector w(d, 0.0);

    for (int ep = 0; ep < epochs; ++ep) {
        Vector grad(d, 0.0);
        for (int i = 0; i < n; ++i) {
            double pred = w[0];
            for (int j = 1; j < d; ++j) pred += w[j] * X[i][j - 1];
            double err = pred - y[i];
            grad[0] += err;
            for (int j = 1; j < d; ++j) grad[j] += err * X[i][j - 1];
        }
        for (int j = 0; j < d; ++j) w[j] -= lr * grad[j] / n;
    }
    return w;
}

    Stochastic Update

    SGD uses one row at a time. The update is noisy but cheap:

    void sgd_step(Vector& w, const Vector& x, double y, double lr) {
    double pred = w[0];
    for (size_t j = 0; j < x.size(); ++j) pred += w[j + 1] * x[j];
    double err = pred - y;
    w[0] -= lr * err;
    for (size_t j = 0; j < x.size(); ++j) w[j + 1] -= lr * err * x[j];
}
    Scale features first. Gradient descent behaves much better when columns have comparable scale.
