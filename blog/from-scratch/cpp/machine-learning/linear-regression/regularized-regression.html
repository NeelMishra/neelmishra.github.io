    Regularized Regression in C++

    July 2026 · 17 min read

    Ridge Regression

    Ridge adds a diagonal penalty to the normal equations. The intercept is usually not penalized.

    Vector fit_ridge(const Matrix& X, const Vector& y, double lambda) {
    int n = (int)X.size(), d = (int)X[0].size() + 1;
    Matrix xtx(d, Vector(d, 0.0));
    Vector xty(d, 0.0);
    for (int i = 0; i < n; ++i) {
        Vector row(d, 1.0);
        for (int j = 1; j < d; ++j) row[j] = X[i][j - 1];
        for (int a = 0; a < d; ++a) {
            xty[a] += row[a] * y[i];
            for (int b = 0; b < d; ++b) xtx[a][b] += row[a] * row[b];
        }
    }
    for (int j = 1; j < d; ++j) xtx[j][j] += lambda;
    return solve_linear(xtx, xty);
}

    Lasso Soft Threshold

    Lasso can be fit with coordinate descent. Each coordinate update applies soft thresholding:

    double soft_threshold(double z, double lambda) {
    if (z > lambda) return z - lambda;
    if (z < -lambda) return z + lambda;
    return 0.0;
}

    A full implementation cycles through features, recomputes partial residuals, and applies this threshold. Elastic Net adds a Ridge term to stabilize correlated features.

    Scale before regularizing. Without scaling, Lasso and Ridge penalize features differently just because units differ.
