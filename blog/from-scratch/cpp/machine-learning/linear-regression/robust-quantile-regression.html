    Robust and Quantile Regression in C++

    July 2026 · 16 min read

    Robust linear variants keep the weighted-sum prediction but change the loss and gradient.

    Huber Gradient

    double huber_grad(double err, double delta) {
    if (err > delta) return delta;
    if (err < -delta) return -delta;
    return err;
}

    Use this clipped gradient in the same gradient descent loop. Large residuals no longer dominate the update.

    Pinball Subgradient

    double pinball_grad(double pred, double y, double tau) {
    return pred >= y ? (1.0 - tau) : -tau;
}

    For quantile regression, replace the squared-error gradient with the pinball-loss subgradient. A model trained with tau = 0.9 estimates the conditional 90th percentile instead of the mean.

    Shared Training Skeleton

    for each epoch:
    for each row:
        pred = dot(w, x)
        grad_error = loss_gradient(pred, y)
        update intercept and weights with grad_error
    Different target, same linear form. OLS predicts the conditional mean, LAD predicts the median, and quantile regression predicts the requested quantile.
