I built something I wish existed when I was learning Data Structures.

Over the past few weeks, I've been writing a deep-dive blog series on Trees — and instead of just explaining the theory, I built fully interactive, step-by-step animations for every single algorithm.

Here's what's live right now:

🌳 **12 posts** covering Trees from fundamentals to advanced patterns
⚡ **Every algorithm is animated** — watch nodes light up, edges trace, and pointers move in real-time
📚 **Dual visualization** — see the tree AND the underlying data structure (call stack, queue, array) side by side
🎛️ **Speed control** — slow it down to 0.3× to understand, speed it up to 4× when you get it
🏹 **Directional arrows** — follow exactly where the algorithm goes at each step

What's covered:
→ In-Order, Pre-Order, Post-Order, Level-Order Traversals (with recursive call stack)
→ BST Search, Insert, Delete (with comparison logs)
→ AVL Rotations — Left, Right, Left-Right, Right-Left (with balance factor tracking)
→ Binary Heaps — Insert, Extract-Min, Heapify (with array representation)
→ Segment Trees — Build, Range Query, Point Update
→ LCA, Tree Diameter, Height Computation
→ Threaded Binary Trees — Construction & Traversal without recursion

I also have a C++ STL series (Vectors, Maps, Iterators, Smart Pointers, RAII) if that's your thing.

No frameworks. No React. Pure HTML/CSS/JS with hand-crafted SVG animations.

Why? Because I believe the best way to understand an algorithm is to *watch it think*.

Check it out → neelmishra.github.io

The series is still growing. If there's a data structure or algorithm you'd love to see animated, drop it in the comments 👇
